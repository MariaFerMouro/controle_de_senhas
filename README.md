Projeto de estudo. Cria um menu onde o usuário escolhe entre fazer login, se cadastrar, entrar no banco de dados (banco_de_dados.txt) e sair do menu.
Opções explicadas do menu:
Login: "Entra no sistema" se o username e a senha estiverem certos.
        Se o username não existir no sistema, o usuário será informado.
        se o username existir e a senha estiver errada, o usuário será informado.

Cadastro: Cadastro um novo usuario com username(não existente) e senha, e apresenta uma sujetão de senha forte.

Acesso ao banco de dados: Só funciona com com a senha do "administrador", dá acesso a todos os usuarios e senhas cadastrados.

Sair: Sai do looping e encerra o programa.
